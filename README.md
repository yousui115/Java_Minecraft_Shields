## ■はじめに
- 新たな盾を追加するModです。
- 予告無しに仕様変更を行う事があります。
- 他Modによる競合は関知しません。
- 作りが荒い為、不具合等が発生する恐れがあります。
- 定期的なバックアップが取れて、不具合に直面しても泣かない強い子が使ってください。

## ■追加盾
- 炎の盾
- 氷の盾
- 黒曜の盾
- 金剛の盾
